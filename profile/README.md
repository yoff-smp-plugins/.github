# yoff-smp-plugins

Внутренняя экосистема кастомных плагинов, коннекторов и инфраструктурных решений, разработанных специально для сети проектов **yoff ✦ smp**.

## 📄 Лицензирование
Все проекты в данном репозитории распространяются под свободной лицензией **Apache License 2.0**. Вы можете модифицировать, распространять и использовать данный код в любых целях, включая коммерческие.

## 🛠 Активные компоненты

| Репозиторий | Спецификация | Стек | Статус |
| :--- | :--- | :---: | :---: |
| **`OnlineDSConnect`** | Асинхронный JDA-мост для динамического вывода сетевого статуса и онлайна в сетку каналов Discord. | Java / Maven | `stable` |
| **`customcmd`** | Оптимизированный обработчик кастомных команд и алиасов ядра. | Java / Maven | `stable` |
| **`JustRTP`** | Легковесный асинхронный плагин случайного телепорта с защитой TPS и поддержкой ванильного WorldBorder. | Java / Maven | `stable` |
| **`YoffAuth`** | Локальная система авторизации на SQLite с сессиями и хешированием SHA-256. | Java / Maven | `stable` |

---

## 🏗 Спецификация окружения

- **Базовая платформа:** Paper / Velocity Proxy `1.21.1` (с поддержкой протоколов вплоть до `1.21.4`)
- **Сборка автоматизации:** Apache Maven 3.8+
- **Версия Java:** Java 21 или выше

## 📌 Стандарты разработки

### Изоляция зависимостей (Shading & Relocation)
Во избежание конфликтов классов (`NoClassDefFoundError` / `ClassCastException`) во время рантайма на сервере, все внешние тяжелые библиотеки **в обязательном порядке** упаковываются внутрь плагина и релоцируются.
